# 프로젝트 진행 상황

## ✅ 완료된 기능

### Frontend (Next.js)

#### 사용자 페이지
- **메인 페이지** (`/[lng]/page.tsx`)
  - 히어로 섹션 + 배경 슬라이드쇼
  - 서비스 혜택 섹션 (6개 주요 혜택)
  - 이용 방법 4단계 프로세스
  - 고객 후기 캐러셀
  - 숙소 안내 섹션
  - 멤버십 & 리뷰 섹션
  - 실제 DB 데이터 표시 (AccommodationsList)

- **서비스 가이드** (`/[lng]/service-guide/page.tsx`)
  - 서비스 특징 설명
  - 4단계 이용 방법
  - 3가지 요금제 (스탠다드/익스프레스/프리미엄)
  - 서비스 지역 안내 (도쿄/오사카/교토/요코하마)

- **숙소 가이드** (`/[lng]/accommodation-guide/page.tsx`)
  - 주요 도시별 숙소 소개
  - 숙소 타입별 안내 (호텔/료칸/에어비앤비/게스트하우스)
  - CarryDrop 제휴 추천 숙소
  - 숙소 연계 서비스 안내

- **멤버십** (`/[lng]/membership/page.tsx`)
  - 3가지 멤버십 플랜 (베이직/프리미엄/VIP)
  - 추가 혜택 안내
  - 자주 묻는 질문

- **배송 신청** (`/[lng]/request-delivery/page.tsx`)
  - 4단계 예약 폼 (출발지/목적지/서비스타입/고객정보)
  - 실시간 요금 계산
  - 유효성 검증

#### 공용 컴포넌트
- **AccommodationsList**: 실제 DB 숙소 데이터 표시
- **FAQSection**: 자주 묻는 질문
- **ReviewsCarousel**: 고객 후기 슬라이드
- **HeroBackground**: 메인 페이지 배경

#### 관리자 페이지
- **대시보드** (`/admin/dashboard/page.tsx`)
  - 주요 통계 대시보드
  - 실시간 현황 모니터링
  - 최근 예약/알림 표시

- **예약 관리** (`/admin/reservations/page.tsx`)
  - 예약 목록 조회/필터링
  - 상태 관리 (대기/결제/배송중/완료)
  - 상세 정보 모달

- **숙소 관리** (`/admin/accommodations/page.tsx`)
  - 숙소 목록 관리
  - 활성/비활성 상태 토글
  - 배송 정보 관리

#### 다국어 지원
- 한국어 완료 (`/locales/ko/common.json`)
- 동적 라우팅 (`/[lng]/...`)

### Backend (NestJS + Kotlin)

#### API 엔드포인트
- **AccommodationController**
  - `GET /api/accommodations` - 전체 숙소 목록
  - `GET /api/accommodations/active` - 활성 숙소만
  - CORS 설정 완료 (localhost:3000)

#### 보안 설정
- Spring Security 구성
- 공개 경로 설정 (`/api/accommodations/**`)
- 관리자 경로 보호

#### 데이터베이스
- PostgreSQL (Neon 클라우드)
- Accommodation 엔티티 완성
- 마이그레이션 스크립트 (V2__Add_admin_entities.sql)

- **해결책**: 전체 경로 사용 시도 중
- **우선순위**: 🟡 Medium

- **상태**: ✅ 해결됨
